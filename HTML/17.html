Favicon

Favaicon is an icon next to the title of the web page. It is an image which shows at the title.It is defined by link element with href.href used to link the image with its extention type i.e. jpg or png. and it has a type which is defined the type of image .

Pathing

when an image or file is located to another and we want to link into it to the existing file the we use the pathing method to link them.and the Path should be exactly matched typed.syntax: foldername/foldername/.../filename.extentionname. and it is case Sensitive.
